Dice (Oct 2021)


Believe it or not, all of those little squares are dice (they have six possible states, so, technically, they are dice). This is done through a dice class, which takes an x and y input for the constructor. Then, a simple for loop creates an array of dice, constantly rolling. The sum total of all the dice is printed in the console log, which I'm not certain that you can see. One difficulty I had was making sure that the for loop worked, because I can't seem to figure it out in javscript. I ended up needing to ask for help. Check out the code here.

Conway's Game of Life! (Nov 2021)


My assignment was to make a game, but who says a game has to have any players? You probably know the rules, any cell with 2 or 3 neighbors lives, and any dead cell with 3 neighbors becomes an alive cell. As it turns out, programming this is no mean feat. Figuring out the status of a cell's neighbors involves a lot of array-based math. It doesn't help that the game of life traditionally takes place on an infinite grid; I had to compromise by making a border of dead cells. I figured it out, and I think the end result is quite impressive. Check out the code here.

Sundial (Dec 2021)


The assignment was simple, no? Just create a clock. To my suprise, It's much more difficult to create a sundial than it would first appear. In a previous version, the sundial was smaller, and didn't fully fill the frame. Calculating the length of the shadow proved to be an endeavor, as none of the extremely complex polynomials I inputted seemed to come out quite right. I resigned myself to failure, and let the project fester for the rest of my senior year in highschool. The next September, I decided to pick the project back up, mostly to flesh out my portfolio. I made the sundial bigger, and the shadow became a right triangle. A simple trigonometric equation gave me a result I was rather happy with. There's some sort of lesson to be learnt here, but this project made me to tired to think about any of that. Check out the code here.
